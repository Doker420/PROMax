# PROMax
Мощный инструмент для автоматизации и массового управления аккаунтами MAX мессенджера.

## Возможности программы

### 🔐 Управление аккаунтами
- Авторизация аккаунтов через QR-код
- Управление несколькими аккаунтами
- Поддержка прокси (HTTP/SOCKS5)
- Просмотр активных сессий

### 👥 Инвайтинг
- Приглашение пользователей по ID в группы/каналы
- Массовый инвайтинг (пачки по 50 ID)
- Выбор аккаунтов для инвайта
- Настраиваемая задержка между пачками

### 📢 Рассылки
- **Рассылка по ID** - отправка сообщений пользователям по их ID
- **Рассылка по чатам** - отправка сообщений в несколько чатов
- Поддержка вложений (фото)
- Рандомизация текста: `{random:вариант1|вариант2|вариант3}`
- Выбор аккаунтов и настраиваемая задержка

### 📊 Парсинг
- **Парсинг участников группы** - сбор ID пользователей из чатов/каналов
- **Парсинг контактов** - сбор всех контактов аккаунта
- Фильтрация (исключить ботов, удалённые аккаунты)
- Сохранение в файл

### 🔥 Прогрев аккаунтов
- Имитация поведения реального человека
- Настраиваемая длительность прогрева (1 час - 1 неделя)
- Вступление в чаты/каналы
- Рандомные сообщения в чатах
- Реакции на посты
- Общение между своими аккаунтами
- Настраиваемые интервалы между действиями

### 👤 Управление профилем
- Смена имени (с поддержкой файлов имён)
- Смена фамилии
- Смена БИО/статуса
- Смена аватарки

### 🔍 Проверка номеров
- Проверка зарегистрирован ли номер в MAX
- Массовая проверка из файла

### 🧹 Чистка чатов

## Рандомизация текста

Используйте конструкцию `{random:слово1|слово2|слово3}` в текстах сообщений для рандомизации:
```
Привет {random:друг|товарищ|коллега}!
```

При отправке случайное слово будет выбрано из списка.
